### 基本说明

本次课程设计基于针对NVM优化的数据结构`FPTree`，实现一个简单的单线程键值存储引擎`FPTreeDB`。我们通过将其包装成一个调用库，供用户程序使用并管理其数据存储，与`LevelDB`的使用方式类似。为了证明其高效性，我们还将其和`LevelDB`进行测试对比。
